# Pulse Studio — Fitness Landing Page

A modern, high-energy landing page concept for a boutique fitness studio. Designed to convert visitors into sign-ups with a bold hero section, clear pricing, and a class schedule preview.

🔗 **Live Site:** [pulse-by-talha-tahir.vercel.app](https://pulse-by-talha-tahir.vercel.app)

## Overview

Pulse Studio is a fictional fitness brand built as a portfolio piece, showcasing a complete landing page experience — from first impression to conversion.

## Features

- **Hero Section** — Full-bleed visual with bold typography and dual call-to-action buttons
- **Class Schedule Preview** — Clean grid layout showing class times, coaches, and intensity levels
- **Pricing Plans** — Three-tier pricing with a highlighted "Most Popular" plan
- **Responsive Design** — Fully optimized for mobile, tablet, and desktop
- **Smooth Interactions** — Hover animations and micro-interactions throughout

## Tech Stack

- **Framework:** Next.js
- **Styling:** Tailwind CSS
- **Deployment:** Vercel

## Design Notes

- Dark theme with a neon accent color for an energetic, modern feel
- Typography pairing chosen to balance bold impact with readability
- Layout built to highlight conversion points (pricing, CTAs) without feeling pushy
